# Codewars Kotlin Solutions

This repository contains my solutions to Codewars coding challenges written in Kotlin.

The solutions are organized by kyu level (difficulty) to track progress and improve problem-solving skills.

## Structure

- 8 kyu — beginner problems
- 7 kyu — easy problems
- 6 kyu — intermediate problems
- 5 kyu+ — advanced problems

## Goal

- Improve Kotlin programming skills
- Practice algorithms and problem solving
- Write clean and readable code
